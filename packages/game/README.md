# 🦞 龙虾对决 - Lobster Battle

深海争霸 · 弱肉强食

## 技术栈

- **前端**: Next.js 14 (App Router) + React + TypeScript
- **状态管理**: Zustand
- **样式**: Tailwind CSS
- **后端**: Next.js API Routes + Socket.io
- **数据库**: SQLite (开发) / PostgreSQL (生产)
- **ORM**: Prisma

## 核心系统

### 1. 大厅系统 (Lobby)
- 玩家信息展示
- 资源显示 (珍珠、贝壳、经验)
- 系统入口导航

### 2. 任务系统 (Tasks)
- **每日任务**: 每日刷新，完成获取奖励
- **主线任务**: 推进游戏进度
- **成就系统**: 长期目标

### 3. 推图系统 (Campaign)
- 章节关卡
- 难度递增
- 首通奖励
- 星级评价

### 4. 副本系统 (Dungeon)
- 限时开放
- 特殊规则
- 稀有奖励
- 挑战次数限制

### 5. 养成系统 (Progression)
- 龙虾等级
- 技能升级
- 装备/外观
- 天赋树

### 6. 战斗系统 (Battle)
- PVE: 玩家 vs AI
- PVP: 玩家 vs 玩家 (实时匹配)
- 回合制策略

## 快速开始

```bash
# 安装依赖
npm install

# 初始化数据库
npx prisma db push

# 开发模式 (需要两个终端)
# 终端 1: 启动 Next.js
npm run dev

# 终端 2: 启动 Socket.io 服务
npm run socket

# 或者使用 tsx (推荐)
npx tsx server.ts
```

### 生产构建

```bash
npm run build
npm start
```
